# Contributing Guide

感谢你对本项目的关注！欢迎贡献代码。

## 如何贡献

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/your-feature`)
3. 提交你的修改 (`git commit -m 'Add some feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建一个 Pull Request

## 代码规范

- 保持代码简洁清晰
- 提交信息使用英文，简明扼要

## 问题反馈

如果发现 bug 或有功能建议，请通过 Issues 提交。
